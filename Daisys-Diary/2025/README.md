# Daisy’s Diary — 2025

This folder contains Daisy’s Diary entries written during the year 2025.

2025 marks the beginning of Daisy’s Diary as a living archive.  
Entries from this year capture the first recorded moments of Daisy’s inner world.

Each sub-folder represents a single day.
Each day contains exactly one diary page.

The diary follows time as it unfolds.
